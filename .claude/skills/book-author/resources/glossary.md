# База знаний: глоссарий терминов

> Источник истины для определений. Дата актуальности: 2026-06.
> Правило для `book-author`: термин объясняется **при первом появлении** в шаге (одно предложение в духе ниже) и считается «занесённым» — дальше можно использовать без повторного определения.

## Веб и API

- **Фреймворк** — библиотека, задающая каркас приложения; вы заполняете его своим кодом (здесь — FastAPI).
- **Веб-сервер (ASGI-сервер)** — программа, которая держит приложение «в эфире» и передаёт ему входящие HTTP-запросы (здесь — Uvicorn/Gunicorn).
- **ASGI** — стандарт связи между async-приложением и сервером; по нему Uvicorn общается с FastAPI.
- **Эндпоинт** — конкретный адрес API (метод + путь), например `GET /health`.
- **Маршрут (route)** — связка «метод+путь → функция-обработчик».
- **Декоратор** (`@app.get(...)`) — обёртка над функцией; здесь регистрирует функцию как обработчик маршрута.
- **JSON** — текстовый формат данных в HTTP; FastAPI сам превращает возвращённый словарь/модель в JSON.
- **OpenAPI / Swagger UI** (`/docs`) — авто-документация, сгенерированная FastAPI из аннотаций и типов.

## Python

- **Виртуальное окружение (venv)** — изолированный набор зависимостей проекта, не пересекающийся с системным Python.
- **Аннотация типа** (`x: int`, `-> dict`) — подсказка о типе; читается людьми, IDE и FastAPI (валидация/доки).
- **`async`/`await`** — асинхронная функция умеет «уступать» управление на время ожидания (БД/сеть), чтобы сервер обслуживал другие запросы.
- **Контекстный менеджер** (`async with`) — конструкция, которая гарантированно открывает и закрывает ресурс (например, сессию БД).
- **dataclass** — класс-контейнер данных с авто-конструктором (используем для доменных сущностей).

## Чистая архитектура

- **DTO / схема** — Pydantic-модель данных запроса/ответа API (слой presentation), напр. `UserCreate`/`UserRead`.
- **Доменная сущность** — бизнес-объект и его правила, не зависящий от FastAPI/БД (слой domain).
- **Репозиторий** — абстракция доступа к данным; **интерфейс** (`Protocol`) в `repositories`, **реализация** в `infrastructure`.
- **Usecase** — один бизнес-сценарий (например «зарегистрировать пользователя»); оркеструет репозитории.
- **Dependency Injection (DI)** — передача готовых зависимостей в код извне; в FastAPI — через `Depends(...)`.
- **`Protocol`** — способ описать интерфейс (набор методов) без наследования; реализация совпадает по сигнатурам.

## Данные и безопасность

- **ORM** — отображение таблиц БД на классы Python (здесь — SQLAlchemy: `UserModel`, `Mapped`, `mapped_column`).
- **Миграция** — версионированное изменение схемы БД (Alembic); применяется `upgrade`, откатывается `downgrade`.
- **Хэш пароля** — необратимое преобразование пароля (bcrypt); в БД хранится только хэш, не сам пароль.
- **JWT** — подписанный токен, подтверждающий личность пользователя без хранения сессии на сервере.

## Тесты и доставка

- **Фикстура (pytest)** — переиспользуемая заготовка для тестов (например тестовый HTTP-клиент).
- **CI (GitLab CI)** — автозапуск линта/тестов/сборки на каждый push.
- **Merge Request (MR)** — запрос на слияние ветки фичи в `main` с ревью и зелёным CI.
- **reverse-proxy** — Nginx перед приложением: терминирует HTTPS и проксирует запросы в контейнер.

## Использование в книге

`book-author`: при первом употреблении термина из этого списка — давать краткое определение (можно дословно). Не плодить синонимы (см. `naming_conventions.md`). Новые термины, которых тут нет, объяснять по тому же принципу.
